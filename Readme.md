## DATA STRUCTURES
Data structures in Java is a method of organizing and storing data so that it can be used more efficiently. 
There are two types of data structures: 
1. Primitive data structures 
2. Non-Primitive data structures. 

The term “primitive data structure” refers to a data structure that only stores one type of data. 

Non-primitive data structures are more complex data structures and are derived from primitive data structures. Non-primitive data structures are divided into two categories:

1. Linear Data Structures
2. Non-linear Data Structures.
